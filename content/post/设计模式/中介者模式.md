# 中介者模式 (Mediator Pattern)

**分类**: 行为型模式 (Behavioral)

## 要解决的问题

假设一个聊天系统有 Alice、Bob、Charlie 三个用户。如果让他们直接通信：

```
  Alice ───→ Bob
    ↑  ←───  ↓
    └── Charlie
```

每个用户需要持有其他所有用户的引用。新增一个用户 David，三个老用户都要改。

**中介者模式**引入一个中介者（聊天室）集中管理通信：

```
         Alice
           │
  ┌────────┼────────┐
  │    ChatRoom     │
  └────────┼────────┘
           │
    ┌──────┴──────┐
    │             │
   Bob        Charlie
```

**多对多网状 → 星形结构**，所有对象只与中介者通信，互不知道对方存在。

## 设计思路

```
  重构前（网状）:              重构后（星形）:

  A ←→ B ←→ C               A ──→ ──┐
    ↺   ↻                       │    │
  D ←→ E ←→ F               B ──→ Mediator ←─ C
                                 │    │    │
                              D ──→ ──┘    │
                                 │         │
                              E ──→ F ────┘
```

**关键角色**：

| 角色 | 类 | 职责 |
|------|-----|------|
| **Mediator** (中介者) | `ChatRoom` | 管理用户列表，路由消息 |
| **Colleague** (同事) | `User` | 只持有中介者引用，不引用其他同事 |

### 核心规则

- User 之间**不互相持有引用**
- User 发消息只调用 `ChatRoom` 的方法
- `ChatRoom` 负责查找收件人并转发
- 新增用户类型或修改路由规则，只改 `ChatRoom`

## 关键代码

### 同事类（User），只持有中介者引用

```cpp
class User {
    ChatRoom* room_;  // 唯一引用的外部对象

    void sendMessage(const std::string& message) const {
        room_->broadcast(name_, message);  // 委托给中介者
    }

    void sendPrivateMessage(const std::string& to, const std::string& message) const {
        room_->sendPrivate(name_, to, message);
    }
};
```

### 中介者（ChatRoom），集中管理交互逻辑

```cpp
class ChatRoom {
    std::vector<User*> users_;

    void addUser(User* user) {
        users_.push_back(user);
        user->setRoom(this);
        broadcast("System", user->getName() + " has joined.");
    }

    void broadcast(const std::string& from, const std::string& message) {
        for (auto* user : users_)
            if (user->getName() != from)
                user->receive(from, message);
    }

    void sendPrivate(const std::string& from, const std::string& to,
                     const std::string& message) {
        for (auto* user : users_)
            if (user->getName() == to) {
                user->receive(from, message);
                return;
            }
        std::cout << "  [Error] User '" << to << "' not found.\n";
    }
};
```

### 客户端

```cpp
ChatRoom chatRoom;

User alice("Alice");
User bob("Bob");

chatRoom.addUser(&alice);  // 自动广播 "Alice has joined"
chatRoom.addUser(&bob);

alice.sendMessage("Hello everyone!");       // 广播给所有人
alice.sendPrivateMessage("Bob", "Hi Bob");  // 私聊
```
