# 组合模式 (Composite Pattern)

**分类**: 结构型模式 (Structural)

## 要解决的问题

文件系统包含文件和文件夹。如果用 if/else 区分处理：

```cpp
// 客户端需要到处写判断
if (isFile(node)) {
    printFile(node);
} else if (isDirectory(node)) {
    for (auto child : node.children) {
        // 递归，但又要判断...
    }
}
```

问题：
- 客户端代码混杂对叶子（File）和容器（Directory）的判断逻辑
- 每新增一种节点类型（如快捷方式、挂载点），所有处理代码都要跟着改

**组合模式**让 File 和 Directory 实现同一个接口，客户端无需区分即可统一操作。

## 设计思路

```
  FileSystemNode (统一接口)
       │
   ┌───┴────────┐
   │            │
  File       Directory  ← 也实现了 FileSystemNode
               │
          ┌────┴────┐
          │         │
        File     Directory
                    │
                  File

  客户端代码：
    forEach(node in tree) {
        node.print();       // 不区分 File 还是 Directory
        total += node.getSize();  // 统一的接口
    }
```

**关键在于**：

| 角色 | 类 | 特性 |
|------|-----|------|
| **Component** (组件接口) | `FileSystemNode` | `getName()`, `getSize()`, `print()` |
| **Leaf** (叶子) | `File` | 没有子节点，`addChild()` 抛异常 |
| **Composite** (组合) | `Directory` | 包含子节点列表，递归委托操作 |

容器操作（`addChild`/`removeChild`）放在 Component 接口中，Leaf 的默认实现抛异常——这是"安全组合模式"的常见做法。

## 关键代码

### 统一接口

```cpp
class FileSystemNode {
public:
    virtual std::string getName() const = 0;
    virtual size_t getSize() const = 0;
    virtual void print(const std::string& indent = "") const = 0;

    virtual void addChild(std::shared_ptr<FileSystemNode> child) {
        throw std::runtime_error("Leaf node cannot have children");
    }
};
```

### 叶子节点（File）

```cpp
class File : public FileSystemNode {
    size_t getSize() const override { return size_; }
    void print(const std::string& indent = "") const override {
        std::cout << indent << "- " << name_ << " (" << size_ << " KB)\n";
    }
};
```

### 组合节点（Directory），递归委托

```cpp
class Directory : public FileSystemNode {
    size_t getSize() const override {
        size_t total = 0;
        for (const auto& child : children_)
            total += child->getSize();  // 递归调用
        return total;
    }

    void print(const std::string& indent = "") const override {
        std::cout << indent << "+ " << name_ << "/\n";
        for (const auto& child : children_)
            child->print(indent + "  ");  // 递归调用
    }
};
```

### 客户端：构建树形结构，统一操作

```cpp
auto root = make_shared<Directory>("project");
root->addChild(make_shared<File>("README.md", 5));
// ... 继续嵌套子目录

root->print("");     // ✅ 一行打印整个树
auto size = root->getSize();  // ✅ 递归计算总大小
```
