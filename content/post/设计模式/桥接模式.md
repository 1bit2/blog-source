# 桥接模式 (Bridge Pattern)

**分类**: 结构型模式 (Structural)

## 要解决的问题

假设你有 2 种形状（圆形、矩形）和 3 种渲染平台（OpenGL、Vulkan、软件渲染）。

如果用继承，每个形状 × 每个平台需要一个子类：

```
               Shape
          /     |     \
  OpenGLCircle  VulkanCircle  SoftwareCircle
  OpenGLRect    VulkanRect    SoftwareRect
```

**M 种形状 × N 种平台 = M × N 个子类**，每新增一种形状或平台，类数量都翻倍。

**桥接模式**把"形状"（抽象）和"渲染"（实现）拆到两个独立继承链，用组合连接它们：

```
  Shape ───has-a──→ Renderer
    │                  │
  Circle           OpenGL
  Rectangle        Vulkan
                   Software
```

**M + N 个类**（而非 M × N），两者可独立扩展。

## 设计思路

```
   ┌─────────────────────────────────────────┐
   │  Abstraction (Shape)                     │
   │  ┌───────────────────┐                   │
   │  │ draw()            │                   │
   │  │ setRenderer(r)    │ ───has-a──→ Renderer (接口)
   │  └───────────────────┘         │
   │         ↑                      ├── OpenGLRenderer
   │    ┌────┴────┐                 ├── VulkanRenderer
   │  Circle  Rectangle             └── SoftwareRenderer
   └─────────────────────────────────────────┘

   继承树: Shape                   继承树: Renderer
   ├── Circle                      ├── OpenGLRenderer
   └── Rectangle                   ├── VulkanRenderer
                                   └── SoftwareRenderer

   新增 Triangle → 只需加一个类
   新增 DirectX   → 只需加一个类
```

## 关键代码

### 实现层接口（Render）

```cpp
class Renderer {
public:
    virtual ~Renderer() = default;
    virtual void renderCircle(float x, float y, float radius) = 0;
    virtual void renderRect(float x, float y, float w, float h) = 0;
    virtual std::string name() const = 0;
};
```

### 抽象层（Shape），持有 Renderer 引用

```cpp
class Shape {
public:
    explicit Shape(std::shared_ptr<Renderer> renderer)
        : renderer_(std::move(renderer)) {}

    virtual void draw() const = 0;

    void setRenderer(std::shared_ptr<Renderer> renderer) {
        renderer_ = std::move(renderer);
    }

protected:
    std::shared_ptr<Renderer> renderer_;
};
```

### 具体形状委托给 Renderer

```cpp
class Circle : public Shape {
public:
    Circle(shared_ptr<Renderer> r, float x, float y, float radius)
        : Shape(std::move(r)), x_(x), y_(y), radius_(radius) {}

    void draw() const override {
        renderer_->renderCircle(x_, y_, radius_);
    }
};
```

### 客户端：同一形状，不同渲染器

```cpp
Circle circle(opengl, 100, 200, 50);
circle.draw();                 // 用 OpenGL 画
circle.setRenderer(vulkan);
circle.draw();                 // 用 Vulkan 画
circle.setRenderer(software);
circle.draw();                 // 用 Software 画
```
